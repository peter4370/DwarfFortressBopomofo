Для v50.3 Steam.

Добавил помощь для переводчиков:

1. CTRL+F1 = активирует автообновление текста в игре, при изменении файла df_rus.csv
2. CTRL+F2 = деактивирует автообновление текста в игре, при изменении файла df_rus.csv
3. CTRL+F3 = отключает перевод
3. CTRL+F4 = включает/обновляет перевод

При нажатии сочетания клавиш должно появиться сообщение о переключении режима.
Иногда оно появляется за окном игры, нужно нажать в нем ОК, чтобы режим заработал.
